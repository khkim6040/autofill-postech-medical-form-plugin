
# Chrome Extension Store Submission Information

## Single purpose
An extension must have a single purpose that is narrow and easy-to-understand.

**Single purpose description:**
This extension is an autofill tool for POSTECH Student Medical Mutual Aid application forms. It stores user-entered information such as student ID, name, and department locally, and automatically fills out forms on the medical mutual aid application page. 


## Permission justification
A permission is either one of a list of known strings, such as "activeTab", or a match pattern giving access to one or more hosts.

**Storage justification:**
Required to store user's personal information (name, student ID, department, account information, etc.) in browser local storage for autofill functionality during medical mutual aid applications.

**ActiveTab justification:**
Required to verify if the current tab is the POSTECH medical mutual aid application page and to execute autofill functionality only on that specific page.


## Are you using remote code?
**No, I am not using Remote code**

**Justification:**
All code is included in the extension package and does not download or execute code from external servers.
